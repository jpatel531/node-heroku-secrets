# node-heroku-secrets

Setting Heroku environment variables from secrets.js

## Getting Started
Install the module with: `npm install node-heroku-secrets`

```javascript
var node_heroku_secrets = require('node-heroku-secrets');
node_heroku_secrets.awesome(); // "awesome"
```
